## Purpose of the App

- Search for Pokemon
- Search Pokemon by type
- Display the details of the respective Pokemon

## Built with

- Mobile Design
- Grid and Flexbox Layout
- React
- React Router DOM
- Fetch API
- PokeAPI
- Dark Mode
- Splash Screen
